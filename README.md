# maven
## 如何使用
```xml
<!-- pom.xml -->
<project>
    ...
    <repositories>
        <!-- gitee -->
        <repository>
            <id>kit101-maven</id>
            <url>https://gitee.com/kit101/maven/raw/master/repository</url>
        </repository>
        <!-- github -->
        <repository>
            <id>kit101-github-maven</id>
            <url>https://raw.githubusercontent.com/kit101/maven/master/repository</url>
        </repository>
    </repositories>
</project>
```
